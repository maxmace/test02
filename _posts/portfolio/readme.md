### Youtube embed format and parameters

The "Full Capability" Syntax
If you want to use every single "bell and whistle" programmed into the site, here is the exact syntax to use in Markdown posts.

This is formatted with carriage returns for readability, which the strip_newlines code in "_includes/youtube.html" should handle perfectly:

{% include youtube.html 
   id="COxMHHi6apU" 
   start="436" 
   end="520" 
   thumb="img/nova.jpg" 
   title="The Supernova Exploration - Orchestral Version" %}
   
#### Parameter Breakdown
Parameter	Purpose	Example
id	The 11-character YouTube video ID.	Example: COxMHHi6apU
start	The time (in seconds) where the video should begin.	Example: 436
end	The time (in seconds) where the video should stop.	Example: 520 (see lower in this readme page for second-counting hack)
thumb	Path to your custom image (relative to the site root).	img/nova.jpg. Need to use img/ root, Example: "img/portfolio/thumb.png"  THIS LINE IS OPTIONAL. WILL PICK UP YOUTUBE THUMB IF NOT SPECIFIED
title	The text that appears in the caption box below the video.	My Video Title


### unique_id scheme 
The unique_id scheme is designed to be a "set it and forget it" safety net. It ensures that if you have three videos on one page, clicking "Play" on the third one doesn't accidentally try to start the first one.

How the unique_id works
In your _includes/youtube.html, we create the ID using this logic:
{{ include.id }}-{{ include.start | default: '0' }}-{{ include.thumb | slugify }}

include.id: This is the YouTube string (like COxMHHi6apU).

include.start: If you play the same video twice but starting at different times, this keeps them distinct.

include.thumb: Since you use custom thumbnails, we "slugify" the filename (turning My Image.jpg into my-image-jpg) to make the ID even more specific.

The id (the YouTube video code) is the only strictly mandatory part. Without it, there is no video to load! The others (start, end, thumb, title) are optional.

## WARNING
Unique ID Collisions: In the rare event you use the exact same video with the exact same start time and the exact same thumbnail twice on one single page, the IDs will collide. If you ever need to do that, just add a dummy character to the title or use a slightly different start time (e.g., 436 and 437) to keep them unique.
